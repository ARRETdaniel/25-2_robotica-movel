Oi pessoal,

Para adiantar e vocês já começarem a pensar, disponibilizei o enunciado do Projeto Final na página principal.

O objetivo principal é tentar aplicar os conceitos e técnicas que estamos vendo para resolver algum problema do "mundo real". Não é necessário fazer nada inovador ou muito complexo, mas é importante que o projeto seja um pouco mais abrangente que os trabalhos práticos. Uma sugestão é pensar em uma tarefa/problema e depois como poderíamos usar um (ou mais) robô para resolver.
